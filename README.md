**RANDOM PASSWORD GENERATOR**

1. PROJECT OVERVIEW

The Random Password Generator is a simple web-based application that helps users generate strong and secure passwords instantly.
It allows users to customize passwords by selecting length and character types such as uppercase letters, lowercase letters, numbers, and special symbols.

The application is built using HTML, CSS, and JavaScript and runs completely on the client side, ensuring privacy and speed.

2. FEATURES

• Generate random and secure passwords
• Customizable password length
• Option to include uppercase letters
• Option to include lowercase letters
• Option to include numbers
• Option to include special characters
• One-click copy to clipboard
• Simple and user-friendly interface
• No backend or database required

3. TECHNOLOGIES USED

HTML – Used for creating the structure of the web page
CSS – Used for styling and layout
JavaScript – Used for password generation logic

4. HOW THE APPLICATION WORKS

User selects the desired password length

User chooses character types (uppercase, lowercase, numbers, symbols)

JavaScript creates a pool of selected characters

Random characters are picked from the pool

Password is displayed on the screen

User can copy the password using the copy button

5. ADVANTAGES

• Generates strong and secure passwords
• Reduces risk of password hacking
• Easy to use and beginner-friendly
• Works offline after loading
• No data storage or tracking
• Fast and lightweight

6. DISADVANTAGES

• Generated passwords are not stored
• No password strength indicator
• Clipboard permission depends on browser
• No advanced security rules (basic version)

7. APPLICATIONS

• Creating secure passwords for websites
• Useful for developers during testing
• Helps students learn JavaScript concepts
• Can be integrated into login systems
• Personal password security use
